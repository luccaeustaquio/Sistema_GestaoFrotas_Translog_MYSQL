# Sistema_GestaoFrotas_Translog_MYSQL
## Banco de Dados relacional para controle de motoristas, veículos e viagens. 
- MySQL

Este projeto utiliza dados fictícios para fins estritamente didáticos e de demonstração técnica, garantindo que nenhuma informação sensível real seja exposta.

# BANCO DE DADOS RELACIONAL PARA EMPRESA FICTÍCIA DE TRANSPORTES

 Criei este banco de dados simples para uma empresa ficticia de transportes de mercadorias. Basicamente este banco de dados vai permitir para a empresa "Translog" o controle de suas operações, como : Quais motoristas utilizaram quais veículos e quais as viagem percorridas mostrando os dados completos dela.
 

ESTRUTURA: 
1- Inciei meu projeto realizando um EER DIAGRAM (Diagrama Entidade e Relacionamento), que me permitiu estruturar visualmente, o banco de dados e as regras de negócio. Então dentro do MySQL Workbench criei minhas entidades com seus atributos, garantindo a integridade referencial entre tabelas através das PKs, FKs, além dos tipos de cardinalidade entre as tabelas.
